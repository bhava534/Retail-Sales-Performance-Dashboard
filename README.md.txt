# Retail Sales Performance Dashboard | SQL + Power BI

## Project Overview
This project analyzes retail sales data to identify revenue trends, top-performing regions, and product performance.

## Tools Used
- SQL (MySQL)
- Power BI

## Key Insights
- South region generated highest revenue
- Beverages contributed 43.02% of total revenue
- Sub-categories A, D, and C were top performers
- Seasonal trends observed in revenue

## Files Included
- Power BI Dashboard (.pbix)
- SQL Queries (.sql)
- Dataset (.csv)
- Analysis Report (.docx)
